# Saraffa Modern Portfolio

Fresh Vite build of the portfolio UI, ported away from Create React App and deprecated packages.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## EmailJS

The contact form reads these Vite environment variables:

```bash
VITE_EMAILJS_SERVICE_ID=
VITE_EMAILJS_TEMPLATE_ID=
VITE_EMAILJS_PUBLIC_KEY=
```
